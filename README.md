# スパジャム道場2023
## ぬけがらみっけ
### アプリ概要
街にあるセミの抜け殻を見つけて，写真を撮り，共有する．  
これをすることにより，セミの抜け殻からセミの生息場所を共有したり，  
「**セミの抜け殻前線**」を作成することができる．
